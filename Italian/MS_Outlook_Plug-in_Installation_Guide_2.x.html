Guida per l’installazione del Plug-in per MS Outlook 2.x

Panoramica

Il Plug-in Sugar per Outlook consente di archiviare email in Sugar, sincronizzare contatti, riunioni, chiamate e compiti, oltre che di creare nuovi record Sugar. Il plugin è disponibile con tutte le edizioni commerciali di Sugar e deve essere installato su ogni computer con Outlook in cui sia necessario il plugin.

La presente documentazione descrive la modalità di installazione del Plug-in Sugar per Microsoft Outlook 2.x che è disponibile per l’utilizzo con Sugar 7.x. Per un elenco completo dei requisiti di piattaforma e degli ambienti supportati, si rinvia alla pagina Piattaforme supportate dai componenti aggiuntivi.

Prerequisiti

Prima di effettuare l’aggiornamento alla versione più recente, si raccomanda di disinstallare eventuali versioni precedenti del Plug-In Sugar per Office. Quando si installa il plugin, tenere presente che occorre effettuare l’accesso al computer come amministratore completo della macchina locale. Tentare di installare sul computer il plugin per un utente non amministratore potrebbe non dare esito positivo.

Nota: Se sulla macchina è già installata la versione di prova gratuita del Plug-in Sugar per Microsoft Outlook, è possibile effettuare l’upgrade alla versione completa del Plug-in Sugar per Microsoft Outlook anziché una nuova installazione.

Disinstallazione delle versioni precedenti del Plug-In Sugar per Outlook

Se sulla macchina è installata una versione precedente del Plug-In Sugar per Outlook, prima di installare la nuova versione, disinstallare quella vecchia.
Disinstallare la versione precedente del Plug-in Sugar per Microsoft Outlook procedendo come segue:

  1. Chiudere il client Microsoft Outlook e gli altri programmi e finestre relativi a Microsoft Outlook.
  2. Nel pannello di controllo di Windows, fare clic su "Installazione applicazioni" o "Programmi e funzionalità" in base alla versione di Windows installata.
  3. Nella finestra che segue, selezionare "Sugar Outlook" quindi fare clic sul pulsante "Disinstalla" o "Rimuovi", in base alla versione di Windows installata.
  4. Viene visualizzato un messaggio pop-up di richiesta di conferma. Fare clic su "Sì" per procedere.
  5. Viene visualizzato un messaggio pop-up che chiede se si desidera mantenere le impostazioni.
    • Nota: Fare clic su "Sì" se si desidera mantenere le impostazioni del plugin. Fare clic su "No" se non si desidera mantenere le impostazioni e si preferisce eliminarle durante la disinstallazione.
  6. Dopo aver rimosso il plugin di Outlook, chiudere la finestra precedente (es. Programmi e funzionalità).

Nota: Se la versione precedente non viene rimossa prima dell’installazione di quella nuova del plugin di Outlook, "SugarOutlook" potrebbe venir visualizzato molteplici volte nella finestra "Installazione applicazioni" o "Programmi e funzionalità".

Controllo della versione di Microsoft Outlook

Perché Outlook corrisponda alla versione del proprio software, è necessario scaricare e installare la versione corretta del Plug-In Sugar per Outlook. Per verificare la versione installata e la versione di bit di Outlook, fare clic sulla scheda File in Outlook, selezionare "Account Office" e successivamente "Informazioni su Outlook". Si prega di notare che un computer a 64 bit può eseguire un’applicazione Outlook a 32 bit, per cui è meglio verificare prima di eseguire l’installazione.

Scaricamento del Plug-In Sugar per Outlook

Una volta verificata la versione e la versione bit di Outlook, è possibile scaricare il file appropriato del plugin effettuando l’accesso alla propria istanza Sugar e facendo poi clic sulla scheda Downloads nel proprio profilo utente.

I file del Plug-In Sugar per Outlook che è possibile scaricare sono:

  • Plug-In Sugar per Outlook 2010 a 32 bit 2.0.0
  • Plug-In Sugar per Outlook 2010 a 64 bit 2.0.0
  • Plug-In Sugar per Outlook 2013 a 32 bit 2.0.0
  • Plug-In Sugar per Outlook 2013 a 64 bit 2.0.0
  • Plug-In Sugar per Outlook 365 a 32 bit 2.0.0
  • Plug-In Sugar per Outlook 365 a 64 bit 2.0.0

Installazione del Plug-In Sugar per Outlook

È ora possibile iniziare a installare il Plug-In Sugar per Outlook una volta scaricato il file corrispondente. In quanto segue sono illustrati i passaggi relativi all’installazione del Plug-In Sugar per Outlook 2013 a 64 bit 2.0.0 a titolo esemplificativo.

  1. Navigare fino alla cartella in cui è stato scaricato il file compresso ed estrarre i contenuti del file compresso.
  2. Fare clic sul file setup.exe per eseguire l’installazione.
    • Nota: Una finestra pop-up con un Avviso di sicurezza potrebbe comparire. Fare clic su "Esegui" per procedere con l’installazione.
  3. Nella fase di benvenuto dell’installazione guidata, fare clic su "Avanti" per avanzare nella procedura di installazione.
  4. Sarà visualizzato un messaggio di avviso indicante che questa versione del plugin è disponibile esclusivamente con versione Sugar 7.x e successive. Fare clic su "Avanti" per procedere.
    • Nota: Non procedere con l’installazione se si sta utilizzando una versione precedente (es. 6.7.x) di Sugar.

  5. Una volta giunti alla fase del License Agreement, selezionare "Accetto" e successivamente "Avanti".
  6. Nella fase "Seleziona la cartella di installazione", lasciare la directory di installazione impostata sul percorso predefinito e selezionare "Solo io". Fare clic su "Avanti".
  7. Nella fase di Conferma installazione, fare clic su "Avanti" per avviare l’installazione sul proprio computer.
  8. Dopo aver completato correttamente l’installazione, fare clic su "Chiudi" per uscire.

Connessione del Plug-In Sugar per Outlook

Una volta conclusa l’installazione, sarà necessario configurare il plugin per Outlook perché si connetta alla propria istanza Sugar in modo che possano comunicare tra loro. Aprire il client di posta Outlook e controllare se, sotto la scheda Home, compaiono le opzioni Sugar (es. Archivia in Sugar, Sincronizza in Sugar, ecc.). Se le opzioni di Sugar non sono presenti, l’installazione potrebbe non essere andata a buon fine.

Attenersi alla seguente procedura per connettere il plugin di Outlook alla propria istanza Sugar:

  1. Fare clic sulla scheda File in Outlook, selezionare "SugarCRM" e quindi scegliere "Opzioni Sugar".
  2. Nella sezione Connessione sotto la scheda Generale, immettere le credenziali di accesso utilizzate per accedere alla propria istanza Sugar e l’URL della propria istanza.
    • Nota: Ad esempio è possibile immettere l’URL della propria istanza come "abc.sugarondemand.com" e "https://" sarà aggiunto automaticamente all’URL dopo la convalida della connessione. Se si desidera immettere il proprio URL di istanza completo, assicurarsi di iniziare l’URL con "https://".

  3. Fare clic sul pulsante Prova connessione per verificare che la connessione sia stabilita con successo.
  4. Fare clic su "OK" se la prova si conclude con successo.

Librerie necessarie

Il Plug-In Sugar per Outlook necessita di diverse librerie per poter funzionare. Il file setup.exe installerà due delle librerie necessarie se del caso. Si prega di notare che l'esecuzione di SugarWordSetup.msi installa solamente il Plug-In Sugar e potrebbe non determinare il funzionamento del plugin se queste altre componenti non sono state installate precedentemente.

Le librerie necessarie sono le seguenti:

  • Microsoft Visual Studio 2010 Tools for Office SE Runtime (inclusa nel file di installazione setup.exe).
  • Microsoft .NET 4.0 Framework (deve essere installata separatamente se non è già presente sul computer).

Risoluzione dei problemi di installazione

Se risulta difficile installare correttamente il Plug-In Sugar, è possibile che le versioni di queste librerie installate sul computer siano obsolete. Per porvi rimedio, occorrerà procedere come segue:

  1. Nel pannello di controllo di Windows, fare clic su "Installazione applicazioni” o "Programmi e funzionalità" in base alla versione di Windows installata.
  2. Selezionare “Microsoft Visual Studio 2010 Tools per Office Runtime" quindi fare clic sul pulsante “Disinstalla" o “Rimuovi”, in base alla versione di Windows installata.
  3. Eseguire nuovamente setup.exe per installare il Plug-In Sugar per Outlook che sostituirà queste librerie con le versioni aggiornate.
