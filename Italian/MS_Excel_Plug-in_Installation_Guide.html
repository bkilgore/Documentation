Guida per l’installazione del Plug-in per MS Excel

Panoramica

Il Plug-in SugarCRM per Microsoft Excel consente di creare, cancellare, aggiornare o visualizzare i dati presenti nella propria applicazione SugarCRM. È possibile visualizzare e/o modificare i dati acquisendo dati dalla Procedura guidata query, oppure è possibile acquisire i dati dai report integrati in Sugar. Il plugin è disponibile con tutte le edizioni commerciali di Sugar e deve essere installato su tutti i computer che eseguono Excel con necessità del plugin.

Passaggi da completare

Se si sta installando il plugin per Excel per un utente, sarà necessario essere connessi come utenti al relativo computer. È inoltre necessario che l’utente disponga di privilegi di amministratore completi per poter scrivere sui file e sulle directory necessari. Si prega di notare che l’installazione su computer del plugin per Excel per un utente non amministratore potrebbe non riuscire. I passaggi sotto riportati illustrano la procedura da seguire per accedere e installare il file compresso necessario per scaricare e installare il Plug-In Sugar per Excel.

Verifica della versione di Microsoft Excel

Perché Excel corrisponda alla versione del proprio software, è necessario scaricare e installare la versione corretta del Plug-In Sugar per Excel. Per verificare la versione installata e la versione bit di Excel è sufficiente controllare quanto riportato alla voce "Informazioni su Excel" o andando su File > Aiuto a seconda della versione di Excel di cui si è in possesso. Si prega di notare che un computer a 64 bit può eseguire un’applicazione Excel a 32 bit, per cui è meglio verificare prima di eseguire l’installazione.

Scaricamento del Plug-In Sugar per Excel

Una volta verificata la versione e la versione bit di Excel, è possibile scaricare il file del plugin in Sugar facendo clic sulla scheda Downloads nel proprio profilo utente.

  • Per Excel 2007, è opportuno scaricare il Plug-In Sugar per Excel 2007.
  • Per Excel 2010 64-bit, è opportuno scaricare il Plug-In Sugar per Excel 2010 x64.
  • Per Excel 2010 32-bit, è opportuno scaricare il Plug-In Sugar per Excel 2010 x86.

Installazione del Plug-In Sugar per Excel

È possibile iniziare a installare il Plug-In Sugar per Excel una volta scaricato il file corrispondente. In quanto segue sono illustrati i passaggi relativi all’installazione del Plug-in Sugar per Excel 2010 x64 a titolo esemplificativo.

  1. Accedere alla propria istanza Sugar, fare clic sul proprio nome nell’angolo superiore destro e quindi selezionare "Profilo".
  2. Fare clic sulla scheda Downloads e selezionare "Sugar Plug-In per Excel 2010 x64".
  3. Navigare fino alla cartella in cui è stato scaricato il file compresso ed estrarre i contenuti del file compresso.
    • Si aprirà una nuova finestra di Gestione risorse contenente un elenco dei contenuti del file compresso.
  4. Fare clic sul file setup.exe per eseguire l’installazione.
    • Una finestra di dialogo con un Avviso di sicurezza potrebbe comparire. Fare clic su "Esegui" se richiesto.

  5. Fare clic su "Avanti" nella finestra di benvenuto per avanzare nella procedura di installazione.
  6. Nella finestra "Seleziona la cartella di installazione", lasciare la directory di installazione impostata sul percorso predefinito e selezionare "Tutti". Fare clic su "Avanti".
  7. Avanzare nella procedura di installazione guidata per completare l’installazione.
    • Se il sistema richiede se consentire al programma di installare il software sul computer, fare clic su "Sì".
  8. Fare clic su "Chiudi".

Connettere il Plug-In Sugar per Excel

Una volta conclusa l’installazione, sarà necessario configurare il plugin per Excel perché si connetta alla propria istanza Sugar in modo che possano comunicare tra loro. La conferma che il Sugar Plug-In Sugar per Excel è stato installato con successo sarà evidente dal fatto che SugarCRM è visibile nella ribbon dei componenti aggiuntivi.

  1. Fare clic su "SugarCRM" sulla ribbon dei Componenti aggiuntivi e selezionare "Impostazioni".
  2. Immettere le credenziali di accesso utilizzate per accedere alla propria istanza Sugar e l’URL della propria istanza. Fare clic su "OK".
    • Si prega di notare che l’URL dell’istanza deve iniziare con "https://’.
  3. Se la connessione va a buon fine, la finestra delle impostazioni SugarCRM dovrebbe scomparire.

Ora se si fa clic su "SugarCRM" sopra a destra in Excel si dovrebbe vedere che si è collegati. Saranno inoltre disponibili più opzioni nel menù SugarCRM come ad esempio "Aggiorna Report", "Seleziona Report", "Procedura guidata query", "Inserire" ecc.

Librerie necessarie

Il Sugar Plug-In per Excel necessita di diverse librerie per poter funzionare. Il file setup.exe installerà due delle librerie necessarie se del caso. Si prega di notare che eseguire "SugarExcelx64.msi" installa solamente il Plug-In Sugar per Excel e potrebbe non determinare il funzionamento del plugin se queste altre componenti non sono state installate precedentemente.

Le librerie necessarie sono le seguenti:

  • Microsoft Office 2010 Primary Interop Assemblies (inclusa nel file di installazione setup.exe).

  • Microsoft Visual Studio 2010 Tools for Office SE Runtime (inclusa nel file di installazione setup.exe).

  • Microsoft .NET 4.0 Framework (deve essere installata separatamente se non è già presente sul computer).

Risoluzione dei problemi di installazione

Durante l’installazione del Plug-In Sugar per Excel potrebbero verificarsi dei problemi. Quanto segue intende fornire alcune utili soluzioni a problemi comuni che potrebbero manifestarsi.

  1. Durante la procedura di installazione l’utente riceve il messaggio "Microsoft Excel 2010 32bit (x86) non è stato trovato su questa macchina. Confermare che si sta utilizzando l’impostazione plugin corretta".

    È possibile che si stia utilizzando il file di download del Sugar Plug-in errato, per cui sarà necessario scaricare il Sugar Plug-In per Excel 2010 x64 e reinstallarlo. Si prega di ricontrollare la sezione Installazione del Plug-In Sugar per Excel nel presente articolo per quanto attiene alle fasi di installazione.
  2. Dopo aver immesso le proprie credenziali di accesso e l’URL dell’istanza Sugar nella finestra delle impostazioni, l’utente visualizza il messaggio “Impossibile connettersi all’URL fornito”.

Fare clic su "OK" per chiudere il messaggio di errore e confermare che è stato immesso l’URL SugarCRM corretto. Verificare che l’URL dell’istanza inizi con ‘https://’. Si prega inoltre di verificare se l’URL della propria istanza reindirizza a un altro indirizzo di destinazione. In tal caso, reimmettere il nuovo indirizzo di destinazione nella finestra delle impostazioni del componente aggiuntivo Excel SugarCRM e provare a riconnettersi nuovamente.
