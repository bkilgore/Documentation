MS Excel Plug-in Installation Guide

Overview

SugarCRM's Plug-in for Microsoft Excel enables you to create, delete, update, or view data that is in your SugarCRM application. You can view and/or edit data by either pulling in data from the built in Query Wizard or you can pull in data from reports that are built in Sugar. The plug-in is available with all commercial editions of Sugar and must be installed on each computer running Excel where the plug-in is desired.

Steps to Complete

If you are installing the Excel plug-in for a user, you will need to be logged in as the user on their computer. The user also needs to have full administrator privileges to write to the necessary files and directories. Please note that attempting to install the Excel plug-in for a non-administrator user on the computer may not be successful. The steps below will go through how to access and install the zip file required to download and install Sugar Plug-in for Excel.

Check Microsoft Excel Version

You must download and install the correct version of the Sugar Plug-in for Excel to match your software's version. In order to verify the installed version and bit version of Excel, check under "About Excel" or via File > Help depending on the version of Excel that you have. Please note that a 64-bit computer may be running a 32-bit Excel application, so it is best to check prior to running the installation.

Download Sugar Plug-in for Excel

Once you have verified the version and bit version of Excel, you can download the plug-in file in Sugar by clicking the Downloads tab in your user profile.

  • For Excel 2007, download the Sugar Plug-in for Excel 2007.
  • For Excel 2010 64-bit, download the Sugar Plug-in for Excel 2010 x64.
  • For Excel 2010 32-bit, download the Sugar Plug-in for Excel 2010 x86.

Install Sugar Plug-in for Excel

You can begin installing the Sugar Plug-in for Excel once you have downloaded the appropriate file. We will go through the steps of installing the Sugar Plug-in for Excel 2010 x64 as an example.

  1. Log in to your Sugar instance, click your name in the top right corner then select "Profile".
  2. Click the Downloads tab and select "Sugar Plug-in for Excel 2010 x64".
  3. Navigate to the folder where the zip file was downloaded to and extract the contents of the zip file.
    • A new Explorer window will open up with a listing of the contents of the zip file.
  4. Click the setup.exe file to run the installation.
    • A Security Warning dialog window may appear. Click "Run" if prompted

  5. Click "Next" on the Welcome window to proceed with the installation process.
  6. In the Select Installation Folder window, leave the installation directory set to the default path and select "Everyone". Click "Next".
  7. Continue through the installation wizard to complete the installation.
    • If you are prompted to allow the program to install the software on your computer, click "Yes".
  8. Click "Close".

Connect Sugar Plug-in for Excel

Once the installation is completed successfully, you will need to configure the Excel Plug-in to connect to your Sugar instance so that they can communicate to each other. You can confirm that the Sugar Plug-in for Excel was installed successfully as you will see SugarCRM displayed in the Add-ins ribbon.

  1. Click "SugarCRM" on the Add-ins ribbon then select "Settings".
  2. Enter in the login credentials used to access your Sugar instance as well as your instance URL. Click "OK".
    • Please note that the instance URL must begin with 'https://'.
  3. If the connection is successful, the SugarCRM Settings window should disappear.

Now when you click "SugarCRM" on the top right of Excel, you should see that you are logged in. You will also be given more options in the SugarCRM menu such as "Refresh Reports", "Select Reports", "Query Wizard", "Insert" etc.

Necessary Libraries

The Sugar Plug-in for Excel requires several libraries to function. The setup.exe file will install two of the necessary libraries if needed. Please note that running "SugarExcelx64.msi" only installs the Sugar Plug-in for Excel and may not result in a functioning plug-in if these other components have not been installed previously.

The necessary libraries are as follows:

  • Microsoft Office 2010 Primary Interop Assemblies (included in setup.exe installer).

  • Microsoft Visual Studio 2010 Tools for Office SE Runtime (included in setup.exe installer).

  • Microsoft .NET 4.0 Framework (must be installed separately if not already present on your computer).

Troubleshooting Installation

There may be times when you run into issues installing the Sugar Plug-in for Excel. We will provide some helpful solutions to common problems you may encounter.

  1. During the installation process, you are prompted with a message that says "Microsoft Excel 2010 32bit(x86) was not detected on this machine. Please confirm you are using the proper setup of plugin".

    You may be using the incorrect Sugar Plug-in download file, so you will need to download the Sugar Plug-in for Excel 2010 x64 and re-install. Please review the Install Sugar Plug-in for Excel section in this article for installation steps.
  2. After entering in your login credentials and Sugar instance URL in the settings window, you see a message that says "Unable to Connect to the URL provided".

Click "OK" to close out the error message and confirm that you entered in the correct SugarCRM URL. Check to make sure that the instance URL begins with 'https://'. Also check to see if your instance URL redirects you to another destination address. If this is the case, please enter in the new destination address into the SugarCRM Excel Add-In Settings window and try to connect again.
