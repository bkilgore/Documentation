MS Outlook Plug-in Installation Guide 2.x

Overview

Sugar's Plug-in for Outlook allows you to archive emails to Sugar , synchronize contacts, meetings, calls, and tasks, as well as create new Sugar records. The plug-in is available with all commercial editions of Sugar and must be installed on each computer running Outlook where the plug-in is desired.

This documentation describes how to install the Sugar Plug-in for Microsoft Outlook 2.x, which is available for use with Sugar 7.x. For a complete list of platform requirements and supported environments, please refer to the Plug-ins Supported Platforms page.

Preconditions

It is recommended that you uninstall any previous versions of Sugar Plug-in for Outlook before you upgrade to the latest version. When installing the plug-in, please keep in mind that you must log into the computer as a full administrator on that local machine. Attempting to install the plug-in from a non-admin user of the computer may not be successful.

Note: If the free trial version of Sugar Plug-in for Microsoft Outlook is already installed on your machine, you can upgrade to the full version of Sugar Plug-in for Microsoft Outlook instead of performing a fresh install.

Uninstalling Previous Versions of Sugar Plug-in for Outlook

If you have a previous version of the Sugar Plug-in for Outlook installed on your machine, please uninstall the old version prior to installing the new version.
Use the following steps to uninstall a previous version of the Sugar Plug-in for Microsoft Outlook:

  1. Close your Microsoft Outlook client as well as other Microsoft Outlook-related windows and programs.
  2. In the Windows Control Panel, click "Add or Remove Programs" or "Programs and Features" depending on your version of windows.
  3. On the following window, select "Sugar Outlook" then click the "Uninstall" or "Remove" button depending on your version of windows.
  4. A pop-up message will display asking for confirmation. Click "Yes" to proceed.
  5. A pop-up message will display asking if you wish to keep the settings.
    • Note: Click "Yes" if you wish to keep the plug-in settings. Click "No" if you do not wish to keep the settings and would like to have it deleted during the uninstall.
  6. Once the Outlook plug-in is removed, close the previous window (e.g. Programs and Features).

Note: If the previous version is not removed prior to installing the new version of the Outlook plug-in, then "SugarOutlook" may appear multiple times in the "Add/Remove Programs" or "Programs and Features" window.

Checking Microsoft Outlook Version

You must download and install the correct version of the Sugar Plug-in for Outlook to match your software's version. In order to verify the installed version and bit version of Outlook, click the File tab in Outlook, select "Office Account", then choose "About Outlook". Please note that a 64-bit computer may be running a 32-bit Outlook application, so it is best to check prior to running the installation.

Downloading Sugar Plug-in for Outlook

Once you have verified the version and bit version of Outlook, you can download the appropriate plug-in file by logging into your Sugar instance then clicking the Downloads tab in your user profile.

The Sugar Plug-in for Outlook files available for download are as follows:

  • Sugar Plug-in for Outlook 2010 32 bit 2.0.0
  • Sugar Plug-in for Outlook 2010 64 bit 2.0.0
  • Sugar Plug-in for Outlook 2013 32 bit 2.0.0
  • Sugar Plug-in for Outlook 2013 64 bit 2.0.0
  • Sugar Plug-in for Outlook 365 32 bit 2.0.0
  • Sugar Plug-in for Outlook 365 64 bit 2.0.0

Installing Sugar Plug-in for Outlook

You can now begin installing the Sugar Plug-in for Outlook once you have downloaded the appropriate file. We will go through the steps of installing the Sugar Plug-in for Outlook 2013 64 bit 2.0.0 as an example.

  1. Navigate to the folder where the zip file was downloaded to and extract the contents of the zip file.
  2. Click the setup.exe file to run the installation.
    • Note: A Security Warning pop-up window may appear. Click "Run" to proceed with the installation.
  3. On the Welcome step of the setup wizard, click "Next" to proceed with the installation process.
  4. A warning message will display indicating that this version of the plug-in is only available for use with Sugar 7.x and higher. Click "Next" to proceed.
    • Note: Do not proceed with the installation if you are using an earlier version (e.g. 6.7.x) of Sugar.

  5. On the License Agreement step, select "I Agree" then click "Next".
  6. On the Select Installation Folder step, leave the installation directory set to the default path and select "Just me". Click "Next".
  7. On the Confirm Installation step, click "Next" to start the installation on your computer.
  8. Once the installation has completed successfully, click "Close" to exit.

Connecting Sugar Plug-in for Outlook

Once the installation is completed successfully, you will need to configure the Outlook Plug-in to connect to your Sugar instance so that they can communicate to each other. Please open up your Outlook mail client and check to see if the Sugar options (e.g. Archive to Sugar, Sync to Sugar, etc.) appear under the Home tab. If you do not see the Sugar options, the installation may not have completed properly.

Use the following steps to connect the Outlook Plug-in to your Sugar instance:

  1. Click the File tab in Outlook, select "SugarCRM", then click "Sugar Options".
  2. In the Connection section under the General tab, please enter in the login credentials used to access your Sugar instance, as well as your instance URL.
    • Note: For example, you can enter in your instance URL as "abc.sugarondemand.com" and the "https://" will automatically be added to the URL after validating the connection. If you wish to enter in your full instance URL, please be sure to begin the URL with "https://".

  3. Click the Test Connection button to confirm if connection is successful.
  4. Click "OK" if test is successful.

Necessary Libraries

Sugar's Plug-in for Outlook requires several libraries to function. The setup.exe file will install two of the necessary libraries if needed. Please note that running SugarOutlookSetup.msi only installs the Sugar plug-in and may not result in a functioning plug-in if these other components have not been installed previously.

The necessary libraries are the following:

  • Microsoft Visual Studio 2010 Tools for Office SE Runtime (included in setup.exe installer)
  • Microsoft .NET 4.0 Framework (must be installed separately if not already present on your computer)

Troubleshooting Installation

If you have trouble with the Sugar Plug-in installing properly, your computer may have outdated versions of these libraries installed. To resolve this, you will need to complete the following steps:

  1. In the Windows Control Panel, click "Add/Remove Programs" or "Programs and Features" depending on your version of windows.
  2. Select "Microsoft Visual Studio 2010 Tools for Office Runtime" then click the "Uninstall" or "Remove" button depending on your version of windows.
  3. Re-run setup.exe to install the Sugar Plug-in for Outlook which will replace these libraries with updated versions.
